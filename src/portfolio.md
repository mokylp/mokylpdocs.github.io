## 👋 潜心写码，保持敬畏

你好，我是一名深耕技术的 **全栈开发者**，目前核心技术栈围绕 **.NET (C#)** 后端服务以及主流前端技术体系。

我不习惯给自己贴上太多繁杂的标签，我更愿意沉浸在代码里，用一行行简洁、健壮的逻辑去解决实际的问题。从数据库设计、接口架构、缓存队列到前端界面的精细交互，我享受构建完整系统并看着它们稳定、高效运转的过程。在这个日新月异的技术时代，我保持着一份低调与专注，用代码见证每一次思考。

## 🛠️ 专业技能 / 核心技术栈

**💻 后端核心与底层机制 (.NET Ecosystem)**
- 扎实的 **.NET / C#** 基础，深入理解 ASP.NET Core 运行机制，熟练运用 **依赖注入 (DI)** 生命周期与定制 **中间件管道 (Middleware)** 处理复杂业务流。
- 精通数据流转与操作，极度熟练使用 **LINQ** 进行优雅且高效的集合操作。
- 熟练掌握多种 ORM 框架，在复杂项目中灵活穿插使用 **Entity Framework Core (EF Core)** 与 **SqlSugar** 实现健壮的数据访问层。
- 熟悉 **ABP Framework**，能够落地领域驱动设计 (DDD)，打造高内聚、低耦合的企业级稳健后台。

**🗄️ 数据库与中间件 (Database & Middleware)**
- 熟练使用底层关系型数据库 **MySQL** 与 **SQL Server**，具备扎实的表结构设计、索引规划与复杂 SQL 慢查询调优能力。
- 擅长引入 **Redis** 进行热点数据缓存、分布式锁以及会话状态管理，显著提升系统并发瓶颈。
- 掌握 **RabbitMQ** 消息队列中间件，利用异步通信实现业务的削峰填谷、服务的高级解耦与状态最终一致性。

**🎨 前端架构与交互 (Frontend Engineering)**
- 拥抱现代前端开发体系，熟练使用 **React** 框架构建数据驱动、组件高复用、交互流畅的单页 Web 应用 (SPA)。
- 深度使用 **TypeScript**，享受强类型系统带来的代码自文档化、重构安全感与极高的工程可维护性。
- 习惯从产品视角出发，打通前后端分离界限，具备独立交付完整 Web 产品的全栈级视野。

**⚙️ 容器调度与自动化 (DevOps & CI/CD)**
- 熟悉日常 **Linux** 服务器底层环境维护与 Shell 脚本基础配置。
- 掌握运用 **Docker** 容器编排，擅长利用轻巧、隔离的虚拟化环境解决“在我的机器上能跑”的部署痛点。
- 熟练配置 **Jenkins**，跨越人肉部署的屏障，独立从 0 搭建从代码拉取、自动化打包到部署上线的流水线 (CI/CD)。
