# 测试基础

欢迎来到软件测试基础模块！这里汇集了测试工程师必备的理论知识和实践方法。

## 📚 学习内容

- [软件测试理论](软件测试理论/) - 测试基础概念、测试类型、测试级别
- [测试方法与技术](测试方法与技术/) - 黑盒测试、白盒测试、测试用例设计方法
- [测试流程与规范](测试流程与规范/) - 测试流程、测试计划、测试报告

## 🎯 学习目标

通过本模块的学习，你将：

✅ 建立完整的软件测试知识体系  
✅ 掌握各类测试方法和技术  
✅ 学会设计高质量的测试用例  
✅ 了解规范的测试流程和文档编写

## 💡 小贴士

> 扎实的测试基础是成为优秀测试工程师的关键。即使在自动化测试盛行的今天，测试思维和方法论依然是我们最重要的武器。💪
